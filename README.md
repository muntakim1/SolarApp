# SolventZ SolarAPP Project

Welcome to the SolventZ Solar App project. This repository contains the source code for the Admin Web Panel, Mobile Application, and Supabase backend configuration.

## 📁 Project Structure

- **`admin-web/`**: Next.js 15+ Admin Dashboard for managing products, orders, surveys, and support tickets.
- **`mobile-app/`**: Expo/React Native application for customers to browse products and request surveys.
- **`supabase/`**: Database migrations, schema definitions, and seed data.
- **`docs/`**: Comprehensive project documentation (organized by role and module).
- **`scripts/`**: Utility scripts for deployment and environment checks.
